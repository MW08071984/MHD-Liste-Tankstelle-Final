# Push-Benachrichtigungen

Diese Version enthält PWA/Browser-Push-Vorbereitung für Android/Samsung und iPhone.

Wichtig für iPhone:
1. App in Safari öffnen.
2. Teilen-Button drücken.
3. "Zum Home-Bildschirm" hinzufügen.
4. App danach vom Home-Bildschirm öffnen.
5. In der App "Push aktivieren" drücken.

Wichtig für Android/Samsung:
1. App in Chrome oder Samsung Internet öffnen.
2. "Zum Startbildschirm hinzufügen" / "App installieren".
3. App öffnen.
4. "Push aktivieren" drücken.

Hinweis:
Lokale Browser-Benachrichtigungen funktionieren direkt. Automatische Server-Pushs zu festen Uhrzeiten brauchen später optional Firebase oder OneSignal.
