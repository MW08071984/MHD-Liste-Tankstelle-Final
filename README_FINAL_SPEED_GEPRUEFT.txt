MHD Kontrolle - Finale geprüfte Speed-Version

Geprüft / enthalten:
- Übersicht lädt MHD-Artikel ohne große Bilddaten.
- Artikelliste lädt ohne große Bilddaten.
- Hochgeladene Bilder werden kleiner komprimiert.
- MHD heute zählt als abgelaufen.
- Fehlende Artikel Fehler wird abgefangen.
- Doppelte Erfassung gleicher Artikel + gleiches MHD wird blockiert.
- MHD manuell oder Kalender.
- Suchfeld Chef/Stationsleitung.
- Backwaren Tagesende + Sonderabschrift MHD/Bruch.
- Normale Abschrift: MHD, Bruch, Eigenbedarf.
- Kontrolle mit Menge 0 erstellt keine Abschrift und entfernt Eintrag.
- Vibration/Signalton soweit Browser/Handy erlaubt.

Hinweis:
Diese Version ändert keine bestehenden Supabase-Daten und löscht keine Artikel.
