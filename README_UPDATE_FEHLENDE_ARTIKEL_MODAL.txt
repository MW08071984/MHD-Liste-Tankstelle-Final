Update: Fehlende Artikel als separates Fenster
- Wird eine EAN/Artikelnummer beim Erfassen nicht gefunden, öffnet sich ein separates Fenster.
- Im Fenster stehen EAN und Feld für Artikelname.
- Button sendet EAN + Name an „Fehlende Artikel“.
- Erfolgreiches Melden gibt Vibration/Feedback.
- Kategorie-Felder wurden in der Oberfläche ausgeblendet; intern bleibt Sonstiges als Standard erhalten.
