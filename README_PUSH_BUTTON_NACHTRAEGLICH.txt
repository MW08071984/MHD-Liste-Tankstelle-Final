Update: Push nachträglich aktivierbar

- Oben im Kopfbereich gibt es jetzt für alle Benutzer den Button "🔔 Push".
- Wenn beim Login versehentlich Nein gedrückt wurde, kann Push darüber später erneut aktiviert/getestet werden.
- Wenn Android/Chrome Push bereits blockiert hat, zeigt die App einen Hinweis, dass Benachrichtigungen in den App-/Browser-Einstellungen erlaubt werden müssen.
- Die automatische 1x-Abfrage beim nächsten Login bleibt unverändert erhalten.
